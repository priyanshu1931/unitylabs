## Installation

Install my-project with npm

```bash
1. Clone the repository.

2. Install the requirements:
    npm install

3. Create .env File:
    PORT=
    IP=
    NODE_ENV=development
    DB_URI=
    JWT_SECRET=
    JWT_EXPIRES_IN=864000000

3. Start the project:
    npm start


Note: Please create the .env file in the root directory.
```
## Tech Stack
